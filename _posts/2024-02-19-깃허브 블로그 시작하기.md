## **맥에서 깃허브 블로그 시작하기**

---

윈도우에서 찍먹해 봤을 땐 블로그 글 하나만 따라가도 어느정도 되는 것 같았는데, 역시 맥은...

> 이 글의 목적은 그저 기록이기 때문에 간략하게 작성되어있음

### 1. Ruby 설치하기

깃허브 블로그 국룰인 지킬이라는 정적 사이트 생성기를 쓰려면 먼저 루비를 다운 받아야 한다.

윈도우 같은 경우엔 <a href="https://rubyinstaller.org/downloads/">RubyInstaller</a>로 바로 원하는 버전을 설치할 수 있지만, 맥은 아니다. 내 경우엔 일단 기본적으로 ruby가 깔려 있었지만 구버전이라 업데이트가 필요했다.

```zsh
$ brew install rbenv

$ rbenv install -l

$ rbenv install 3.x.x
```

`rbenv` 라는 루비 버전 관리자를 먼저 설치하고, 설치 가능한 ruby 버전의 리스트를 출력한 뒤 거기서 골라 설치하면 된다. 나는 `2.6.x`에서 `3.2.3` 버전으로 업데이트 했다. `2.5.x` 이상이기만 하면 괜찮다는 말이 있었지만 그냥 최신 권장 버전으로 업데이트 하였다.

> `ruby -v` 를 터미널에 입력했을 때 계속 구버전으로 뜨는 문제
{: .prompt-warning}

```zsh
$ rbenv init
eval "$(rbenv init - zsh)"
```

환경 변수를 설정 해줘야 한다. `rbenv init`을 했을 떄 나오는 코드를 `~/.zshrc`에 넣어주면 된다.  
`vim`으로 저 파일 열고, `i`눌러 INSERT 모드로 바꾼 뒤, 붙여 넣고, ESC, `:wq`로 저장하고 빠져나오기.

### 2. Jekyll 설치하기

```zsh
$ gem install jekyll
$ gem install bundler
```

크게 번거롭지 않다. `bundler`는 gem들의 의존성을 관리해주는데 여기서 gem은 ruby에서의 라이브러리 같은 것이라고 이해하면 된다. 잘 설치 됐는지 확인도 해주자.

### 3. 테마 정하기 (급완결)

처음엔 <a href="https://jekyllthemes.io/free">테마 사이트</a>에서 골라 zip 파일을 직접 다운 받고 설정하는 방식을 따르려고 했는데 지금 사용중인 이 테마가 꽂혀서 **<a href="https://chirpy.cotes.page/posts/getting-started/">chirpy doc</a>** 이걸 보고 차근차근 따라했다. 내가 선택한 건 chirpy starter로 메인 레포지토리랑 비교하면 빠진 파일들이 많긴 한데, 크게 커스터마이징 하고 싶은 것도 없고 신경 쓸 여력도 안 돼서 그냥 이렇게 사용하기로 결정.

저 Chirpy 공식 문서가 정말 친절하고 자세해서 이 단계부턴 저것만 그대로 따라가도 된다. 초기 설정, 댓글 기능, 파비콘, 문법 등 진짜 다 써 있다. 힘들기도 하고. 그래서 이 글은 일단 여기서 마치고 나중에 더 쓸 일이 있으면... 또 쓰겠다.
